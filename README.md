# Mykola — Static Portfolio (Dark by default)

Run locally with Python (no Node required):

```bash
cd mykola_portfolio_static
python -m http.server 8000
# → http://localhost:8000
```

- Edit `script.js` → `PROFILE` and `PROJECTS`.
- Put your media in `/assets` and update `mediaSrc`.
- Contact section shows your email: `whateverkaicodes@gmail.com`.

Deploy easily with GitHub Pages or Netlify (drag & drop).
